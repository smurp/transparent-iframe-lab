Purpose:
using only CSS make IFRAMEs which are transparent to clicks but which themselves contain clickable elements
Conclusion:
"semi-transparent" IFRAMEs are not possible with pure CSS
Details:
The only way to make an IFRAME "transparent" to clicks is to set "pointer-events: none" on the IFRAME itself, not merely on the HTML and/or BODY tags within it. When the whole IFRAME is "transparent" then no item within it can be made "opaque" (eg clickable).
